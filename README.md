# Banglow - Premium Real Estate Allotment Portal

Live Demo: [https://banglow.vercel.app/](https://banglow.vercel.app/)

## Overview
Banglow is a high-end, visual-first practice project designed to showcase a modern digital catalog and allotment booking system for signature luxury residences. With a clean, premium black-and-cream aesthetic, the platform represents a bespoke booking portal tailored to upscale enclaves and architecture showcases.

## Core Features & Workflow

### 1. Flagship Collections Showcase
* **Visual Catalog Grid:** Explores premium apartments and towers in a clean, high-contrast, symmetrical grid layout.
* **Responsive crop states:** Preserves vertical architectural outlines, preventing cropped building crowns on mobile and desktop viewports.

### 2. Architectural Art & Sketch Gallery
* **Dynamic Visual Showcases:** Loads specific sketch blueprints and detailed architectural line-drawings (`ART-1` through `ART-6`) dynamically for each individual building.
* **Interactive Zoom Loupe:** Implements a hardware-accelerated zoom-on-hover (on PC) and touch-to-zoom (on Mobile/Tablet) magnifying loupe to inspect fine sketch details closely.
* **Branded Downloader:** Provides custom downloading capability that rebrands and exports blueprint assets with client-facing filenames (e.g., `banglow-{property}-art-{index}.jpeg`).

### 3. Smart Allotment & Unit Reservations
* **Interactive Pricing Sheet:** Displays real-time breakdown of prices, dimensions, floor allocations, and booking deposit percentages.
* **Mobile-Responsive Booking Steppers:** Utilizes space-saving indicators that adapt layout alignments and compact text labels responsively.

### 4. Custom Scheduling Calendar
* **Visual Date Matrix:** Integrates custom calendars to pick a meeting day.
* **Interactive Time Chips:** Replaces basic browser select lists with a clean grid of selectable time slots for a seamless reservation experience.

### 5. Simulated Checkout Gateways
* **Simulated Sandbox:** Simulates payment wires (Bank transfer or Mobile MFS like bKash/Nagad) with reference validation logs and automated countdown receipts.
